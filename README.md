# PROJECT OVERVIEW: THE POWER OF AUTONOMOUS SYSTEMS

## PROJECT I: S3-LAMBDA: THE PULSE OF EVENT-DRIVEN MEDIA

This project focuses on building an automated image processing pipeline using AWS services. By integrating S3, Lambda, EventBridge, and CloudWatch, I developed a system that automatically resizes images upon upload. It demonstrates the power of serverless computing and event-driven architecture, where processes are triggered dynamically without manual intervention.

## PROJECT II: AWS LEX: THE VOICE OF THE HOSPITALITY

The second project involves designing an intelligent chatbot for hotel booking using Amazon Lex. This system simulates a real-world receptionist, guiding users through room selection, booking details, and confirmation. It highlights the integration of conversational AI with cloud services to create interactive and user-friendly applications.
